## 3. 常用指令

```bash

\bin\php\php-7.3.33\php.exe \bin\composer\composer-1.10.10\composer.phar -V

# 建立專案
\bin\php\php-7.3.33\php.exe \bin\composer\composer-1.10.10\composer.phar create-project my-project

# 安裝Laravel
\bin\php\php-7.3.33\php.exe \bin\composer\composer-1.10.10\composer.phar install

# 更新Laravel
\bin\php\php-7.3.33\php.exe \bin\composer\composer-1.10.10\composer.phar update
```
